NLP-based sentiment analysis project using TF-IDF and Logistic Regression to classify text as positive, negative, or neutral
# 📊 Sentiment Analysis using NLP

## 📌 Overview

This project performs sentiment analysis on text data and classifies it as **positive, negative, or neutral** using Natural Language Processing (NLP).

## 🚀 Features

* Text preprocessing and cleaning
* TF-IDF vectorization
* Logistic Regression model
* Sentiment prediction (positive / negative / neutral)
* Simple command-line testing

## 🛠️ Tech Stack

Python, Pandas, Scikit-learn

## 📂 Dataset

Custom dataset with labeled text:

* text
* sentiment

## ▶️ How to Run

```bash
python main.py
```

## 📊 Output

* Classification Report
* Predicted sentiment for custom input

## 💡 Example

Input: "I hate this app"
Output: negative

## 🎯 Conclusion

This project demonstrates how NLP techniques can be used to analyze human emotions from text data.
